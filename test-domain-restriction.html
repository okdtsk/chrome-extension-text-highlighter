Test Scenarios

This page contains the text "days ago" multiple times to test domain restriction.

This post was created 5 days ago.

Example 2

Example 3

Just days ago, we launched this new feature.

How to Test

  1. Open the extension popup
  2. Add a new highlight rule for "days ago"
  3. In Domain Restriction field, enter a pattern like:
    • ^file://.*test-domain.*\.html$ - Only this test file
    • ^https?://.*\.okdtsk\.dev - Only okdtsk.dev domains
    • ^https://github\.com/.* - Only GitHub
  4. Save the rule and check if highlighting works based on the domain pattern

Expected Behavior

The text "days ago" should only be highlighted when the current page URL matches the domain pattern regular expression.
